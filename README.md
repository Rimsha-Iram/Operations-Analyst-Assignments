# Operations Analyst Assignments

This repository contains solutions for two assignments completed as part of an Operations Analyst assessment:

## Assignment 1
- Analyzed customer order data using Google Sheets.
- Calculated revenue, total units, average order value, and category insights.
- Used advanced formulas such as SUMPRODUCT, SUMIFS, INDEX-MATCH, TEXTJOIN, UNIQUE, and array formulas.
- Focused on accuracy, automation, and structured data analysis.

## Assignment 2
- Optimized packaging for pick & pack orders.
- Selected the most suitable box for each order considering item dimensions, quantity, stacking, and efficient fit.
- Demonstrated logical reasoning and operational decision-making.

**Skills Highlighted:**
- Google Sheets formulas & functions
- Data analysis & reporting
- Operational optimization & problem solving
